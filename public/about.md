## 关于 md-editor-v3

Markdown 编辑器，vue3 版本，使用 jsx 和 typescript 语法开发，支持切换主题，支持 prettier 美化文本，支持图片粘贴上传，裁剪上传，支持在 tsx 项目使用。

## 同系列

- react 版本，[md-editor-rt](https://github.com/imzbf/md-editor-rt)，更新比较缓慢（作者有点忙）
- angular 版本，小伙伴开发中
